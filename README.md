# App-Clima (Vanilla JS)

### Description
> **An example developing with webpack to create an app with pure JS, this app consults an API and gives the weather from the location that you consulted**

## Technologies employed

### Front-End
	
	1) Html5
    2) Css3
    
	

### Back-End

	1) NodeJS
    2) Api-Weather
